# Responsive Login and Signup Pages

This project contains simple, clean, and responsive login and signup pages created using pure HTML and CSS. The pages feature modern gradient backgrounds, form validation styling, and are suitable for web development portfolios or beginner frontend practice.

## Features
- Responsive design for different screen sizes
- Modern gradient backgrounds and form focus effects
- Clean and semantic HTML structure
- No external dependencies required

## Usage
1. Download or clone the repository.
2. Open `login.html` or `signup.html` in any web browser.
3. Use the forms as a front-end template for your projects.

## How to Run
No build or installation required. Just open the HTML files in your browser.
